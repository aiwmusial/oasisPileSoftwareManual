Oasys Pile is a software that allows you to calculate the pile load capacity, settlement and the displacement of the surrounding ground. It is a very useful software, so let’s learn how to get the most of it...

LET’S GET STARTED...

Prepared for Pile version 19.7

You can calculate pile capacity, settlement or both. Let’s try it out... Choose the analysis, about which you would like to learn more and press ok!

Three options for the capacity calculations are avaliable: Working Load, Design Resistance and Code Based. Have a look at what is the best option for your design and press ‘Next>’

WORKING LOAD

Input global FOS or partial factors as required, then click ‘Finish’ to get to the next stage.
